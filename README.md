# AutocompleteDiscordQuest
A Vencord userplugin that allows you to complete the recent discord quest that you selected, Vibecoded. Do expect issues. Based off the gist by @aamiaa


# HOW TO USE!!!

First, read [The official Vencord docs about Userscripts](https://docs.vencord.dev/installing/custom-plugins/) and proceed by the instructions.
The second step is to insert the folder from this repository into "/src/userplugins" (You may need to create the folder)
Third, build and inject your vencord install.

If you see a button saying to complete the quest after you start one in the triple dots (context menu), you succeeded. Press the button to complete the quest.


Enjoy!
